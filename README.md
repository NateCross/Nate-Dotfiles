# Nate-Dotfiles

Syncing my personal configs.
This init.vim is intended to be used with Neovim and VSCode.

## Prerequisites

- Win 10 (until I get to use Linux)
- Nodejs + Choco
- Neovim Nightly
- Python 3.10 (and pip)
- fzf + bat
- Preferably use Windows Terminal + Nerd Fonts
- Git
